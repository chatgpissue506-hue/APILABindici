Lab Test API Documentation

Version 1.0.0 | Base URL: http://localhost:5050

GET /api/labtest
Get all lab test data from the stored procedure
GET /api/labtest/patient/{patientId}
Get lab test data for a specific patient
GET /api/labtest/daterange?startDate={date}&endDate={date}
Get lab test data within a specified date range
GET /api/labtest/filter?patientId={id}&startDate={date}&endDate={date}&practiceId={id}
Get lab test data with flexible filtering options
